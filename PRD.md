# PRD: 팀 회의 안건 정리 도구

## 1. 한 줄 설명
팀원 누구나 회의 전에 안건을 빠르게 등록하고, 회의 중 실시간으로 정리할 수 있는 웹 도구

---

## 2. 대상 사용자
- 우리 팀 10명 (외부 사용자는 고려하지 않음)

---

## 3. 핵심 기능

| # | 기능 | 설명 |
|---|------|------|
| 1 | 안건 등록 | 제목, 담당자, 소요 시간을 입력해 안건을 추가 |
| 2 | 안건 순서 변경 | 드래그 앤 드롭으로 논의 순서를 조정 |
| 3 | 회의 진행 체크 | 각 안건을 "논의 중 / 완료 / 보류" 상태로 표시 |
| 4 | 결과 메모 | 안건별로 결정 사항과 액션 아이템을 간단히 기록 |
| 5 | 회의록 내보내기 | 완료된 회의 내용을 텍스트로 복사하거나 파일로 저장 |

> 실시간 공동 편집은 구현 복잡도가 높아 이번 버전에서 제외

---

## 4. 안 만들 것 (범위 밖)
- 로그인 기능 (인증 없이 바로 사용)
- 사용자 계정 관리
- 실시간 다중 사용자 동시 편집 (협업 기능)
- 캘린더 연동 또는 일정 예약
- 모바일 앱 (웹 브라우저만 지원)
- 외부 서비스 연동 (Slack, Notion, Google 등)

---

## 5. 사용자 흐름

1. **회의 생성**: 사용자가 회의 제목과 날짜를 입력해 새 회의를 만든다
2. **안건 등록**: 팀원이 논의할 안건을 제목 · 담당자 · 예상 시간과 함께 추가한다
3. **순서 조정**: 회의 시작 전 안건의 우선순위에 맞게 순서를 드래그로 정렬한다
4. **회의 진행**: 각 안건을 클릭해 상태를 변경하고 결정 사항 · 액션 아이템을 메모한다
5. **회의록 저장**: 회의 종료 후 전체 내용을 복사하거나 파일로 내보내 팀과 공유한다

---

## 6. 성공 기준

| 지표 | 목표 |
|------|------|
| 안건 등록 소요 시간 | 건당 30초 이내 |
| 회의록 내보내기까지 클릭 수 | 3번 이내 |
| 페이지 로딩 속도 | 2초 이내 |
| 첫 사용자 재방문율 | 1주일 내 60% 이상 |
| 모바일 브라우저 정상 동작 | 주요 기능 100% 작동 |
| 회의 전 안건 정리 시간 | 3분 이내 완료 |
